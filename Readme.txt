# HOW TO RUN
To execut the application you have 2 options, using a file or using a comand line

To execute the app with a file input run: 

java -jar leaderboard-0.0.1-SNAPSHOT.jar -f path_to_file

To execute the app for command line interaction run: 

java -jar leaderboard-0.0.1-SNAPSHOT.jar -c
